Description------------------------------------------------------------------------------------------
This style file is designed for compiling book notes as part of the body text.
The "remember picture" feature automatically calculates coordinates, eliminating the need for manual adjustment of note positions.
The main packages used in chuushaku.sty are TikZ, amsmath, framed, and calc.
